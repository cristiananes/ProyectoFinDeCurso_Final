Presentación del Proyecto

Resumen visual del proyecto y guion narrado de la defensa.

  • Título: "MiAlquiler – Plataforma Integral de Gestión de Alquileres"
  • Presentador: Cristian Anes García (Alumno DAW)
  • Notas: Saludo inicial y breve introducción.

  • Empresa PropTech tipo SaaS con modelo de suscripción.
  • Necesidad: digitalizar gestión de alquileres.
  • Diferenciación por enfoque local y facilidad de uso.

  • Organigrama de startup con áreas de Desarrollo, Diseño y Gestión.
  • Funciones clave de cada área.
  • Escalabilidad futura según crecimiento.

  • Alta demanda de pagos automatizados y contratos digitales.
  • La gestión manual produce morosidad y poca transparencia.
  • MiAlquiler cubre esta brecha con mayor eficiencia.

  • Mercado PropTech global ~€15.2B; en España ~€1.8B.
  • Proyección: 50 clientes primer año, 2000 en 5 años.
  • Financiación propia y posibles ayudas públicas.

  • Plataforma web integral para propietarios, inquilinos y administradores.
  • Automatiza pagos, contratos y comunicación directa.
  • Dashboard con reportes en tiempo real.

  • Arquitectura MVC con Spring Boot.
  • Frontend con Thymeleaf y Bootstrap.
  • Base de datos MySQL normalizada.

  • Metodología iterativa por fases.
  • Análisis, desarrollo backend, frontend y pruebas.
  • Unas 12-13 semanas de trabajo efectivo.

  • Despliegue en entorno local con Spring Boot.
  • Preparación para migrar a la nube.
  • Pruebas funcionales y unitarias con usuarios reales.

  • Diagramas UML y guía de usuario.
  • Código en repositorio Git con control de versiones.
  • Checklist de requisitos cumplidos.

  • Stack tecnológico consolidado y costes asumibles.
  • Cumplimiento de obligaciones fiscales y RGPD.
  • Ayudas para emprendedores como ENISA.

  • Hitos semanales de seguimiento.
  • Indicadores de calidad y accesibilidad.
  • Gestión de incidencias mediante Git.

  • MiAlquiler integra todo el ciclo de alquileres.
  • Proyecto viable con opciones futuras (app móvil, microservicios).
  • Agradecimientos al tribunal y turno de preguntas.

Guion Narrado

Se expone de forma narrada el contexto empresarial, la problemática detectada y la solución técnica desarrollada. Cristian Anes García describe cómo MiAlquiler digitaliza la gestión de propiedades, contratos y pagos, y aborda aspectos de despliegue, pruebas y futuras mejoras.
